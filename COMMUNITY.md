## 📦 Templates

Get started in minutes with our template repository:

```bash
git clone https://github.com/Damgeed/memory-bridge-templates
cd memory-bridge-templates
```

Available templates:
- **langgraph-customer-support** — AI support agent with persistent memory
- **autogen-research-team** — 3-agent research collaboration
- **fastapi-memory-quickstart** — Bare-bones FastAPI integration
