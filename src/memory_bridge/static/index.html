🗄️ Where does my data actually go?
In production, to your own PostgreSQL database.

Every memory is a row in a table you own. No vendor lock-in — self-host with SQLite, or scale up to PostgreSQL. Export, query, delete, audit: it's your data.
🧩 How do I connect ChatGPT / Gemini / local models?
Memory Bridge is an API — not a plugin. The pattern is always the same:

Query → Inject → Call LLM → Store

1. Before your LLM call, GET /memories/query for relevant context
2. Inject context into the system prompt
3. Call ChatGPT, Claude, Gemini, Ollama — anything with an API
4. After, POST /memories with what was learned

Python SDK and framework adapters for LangGraph, AutoGen, and CrewAI automate the loop.

The data model.

Four core fields. Everything else is metadata.

📊 Memory Table Schema

FieldWhat it storesExample
agent_idWhich agent wrote this memoryhenry, code-reviewer, assistant
session_idWhich conversation/task this belongs tosession-1, deploy-prod-2024
keyThe kind of fact (like a filename)user_preference, architecture_decision
valueThe actual content (string or JSON){"theme": "dark"}, "use OAuth2"
tagsLabels for filtering/searching["user-profile", "important"]
ttl_secondsOptional expiry (auto-cleanup)86400 (24h), null (permanent)

🔌 Works with every LLM + framework

Memory Bridge sits between your code and any LLM. The integration pattern is the same for all of them.

OpenAI / ChatGPT Anthropic / Claude Google Gemini Ollama (local) llama.cpp OpenRouter LangGraph AutoGen CrewAI
# Python SDK — the "Query → Inject → Call → Store" pattern
>>> from memory_bridge_client import Client

>>> async with Client(url="https://your-instance.com", api_key="...") as mb:

    # 1. Before calling the LLM, get context
    ctx = await mb.query_memories(agent_id="henry", tags=["user-profile"])

    # 2. Inject context into the system prompt
    prompt = f"You already know: {ctx['entries']}"

    # 3. Call any LLM (OpenAI, Claude, Ollama...)
    response = await openai.chat.completions.create(prompt=prompt)

    # 4. Store what was learned
    await mb.create_memory(session_id="s1", agent_id="fred", key="plan", value=response)
