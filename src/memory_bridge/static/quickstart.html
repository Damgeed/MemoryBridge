Get Started in 3 Steps

From zero to your first stored memory in under 60 seconds.

1

Generate an API Key

Create your first API key from the dashboard or generate one instantly via the admin API using the demo key below.

# Generate a new API key via the admin API
$ curl -X POST https://memory-bridge-app-production.up.railway.app/admin/api-keys \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"name": "My Quickstart Key"}'
Open Dashboard
2

Try the MCP Integration

Add Memory Bridge to your Claude Desktop or any MCP-compatible client. Configure your claude_desktop_config.json:

# ~/Library/Application Support/Claude/claude_desktop_config.json
{
"mcpServers": {
"memory-bridge": {
"command": "npx",
"args": ["-y", "@memory-bridge/mcp"],
"env": { "MEMORY_BRIDGE_API_KEY": "YOUR_API_KEY" }
}
}
}
Replace YOUR_API_KEY with the key you generated in Step 1. Restart Claude Desktop and the memory tools will be available automatically.
View All Integrations
3

Store Your First Memory

Use curl — or any HTTP client — to store a memory and then query it back.

# Store a memory
$ curl -X POST https://memory-bridge-app-production.up.railway.app/memories \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
"session_id": "quickstart",
"agent_id": "claude",
"key": "greeting",
"value": "Hello from Memory Bridge!"
}'

✅ Expected response — a 201 Created with the stored memory:

# Response: 201 Created
{
"id": "mem_abc123",
"session_id": "quickstart",
"agent_id": "claude",
"key": "greeting",
"value": "Hello from Memory Bridge!",
"created_at": "2025-01-15T10:30:00Z"
}

Now query it back:

# Query the memory back
$ curl -s "https://memory-bridge-app-production.up.railway.app/memories?agent_id=claude&session_id=quickstart" \
-H "Authorization: Bearer YOUR_API_KEY"

📡 Quick API Reference

Method Endpoint Description
POST /memories Store a new memory
GET /memories?agent_id=<id> Query memories by agent
GET /memories/<memory_id> Get a single memory by ID
DELETE /memories/<memory_id> Delete a specific memory
POST /admin/api-keys Generate a new API key
GET /health Health check endpoint
