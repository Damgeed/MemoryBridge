Frequently asked questions

🛠️ Can I self-host for free?
Yes! Memory Bridge is MIT licensed. Self-host with Docker Compose on any machine. No limits, no restrictions.
⚠️ What happens if I exceed my tier?
API calls above the rate limit return 429. We store all memories until you hit the cap, then new storage fails. Upgrade to unlock higher limits.
🔄 Can I switch tiers?
Yes, upgrade or downgrade anytime. Changes apply immediately. Credits are prorated.
🗄️ How is data stored?
Your data lives in your PostgreSQL database (our cloud or self-hosted). You can export, query, and delete it freely.
🏢 Is there an Enterprise plan?
Yes, contact us for custom pricing, dedicated infrastructure, and SLA guarantees.
