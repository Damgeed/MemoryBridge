🚀 Setup Guide

Get started in 3 steps. Your API key goes in the Authorization header.

1

Install

Install the Python package via pip.

$ pip install memory-bridge
2

Start the server

Run the server locally. It starts on http://localhost:8000.

$ memory-bridge
3

Store your first memory

Use your API key to authenticate. Replace YOUR_API_KEY below.

$ curl -X POST http://localhost:8000/memories \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"session_id":"my-session","agent_id":"henry","key":"hello","value":"Hello Memory Bridge!"}'
