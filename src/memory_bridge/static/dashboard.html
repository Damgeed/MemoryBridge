🚀 Setup Guide

Get started in 2 steps. Your API key goes in the Authorization header.

1

Run with Docker

Pull the image and start Memory Bridge on http://localhost:8000.

$ docker run -p 8000:8000 memorybridge/memory-bridge
2

Store your first memory

Replace YOUR_API_KEY with your key from above.

$ curl -X POST http://localhost:8000/memories \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"session_id":"my-session","agent_id":"henry","key":"hello","value":"Hello Memory Bridge!"}'
