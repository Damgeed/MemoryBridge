OpenAI Agents SDK Integration

Add persistent memory to any OpenAI Agent with a single tool registration. Your assistants remember context across conversations and sessions.

Overview

The MemoryBridgeTool adapter for the OpenAI Agents SDK registers as a standard agent tool. Once added to your agent's tool list, it can store and retrieve memories via the Memory Bridge API. The agent can call store_memory and recall_memory functions naturally during conversations.

✨ Note: The OpenAI Agents SDK adapter is located at memory_bridge.adapters.openai_agents_adapter and provides a ready-to-use MemoryBridgeTool class.

Installation

bash
pip install memory-bridge openai-agents

Quick Start

1

Import and create the memory tool

Create a MemoryBridgeTool instance that exposes memory operations to your agent.

Python
from memory_bridge.adapters.openai_agents_adapter import MemoryBridgeTool
from agents import Agent, Runner

memory_tool = MemoryBridgeTool(
    api_key="mb_your-api-key-here",
    base_url="http://localhost:8000",
    session_id="my-agent-session"
)
2

Create an agent with the memory tool

Pass the memory tool in your agent's tools list.

Python
agent = Agent(
    name="Assistant",
    instructions="You have persistent memory. Use store_memory to save facts and recall_memory to retrieve them.",
    tools=[memory_tool]
)
3

Run the agent

The agent can now store and retrieve memories during conversations.

Python
result = await Runner.run(
    agent,
    input="Remember that my name is Alice and I prefer dark mode."
)
print(result.final_output)

# Later, in a new conversation:
result2 = await Runner.run(
    agent,
    input="What do you remember about me?"
)
print(result2.final_output)
# The agent recalls: "Your name is Alice and you prefer dark mode."

Complete Working Example

A fully runnable OpenAI Agent with Memory Bridge persistence:

Python
import asyncio
from memory_bridge.adapters.openai_agents_adapter import MemoryBridgeTool
from agents import Agent, Runner

async def main():
    # Create the memory tool
    memory = MemoryBridgeTool(
        api_key="mb_your-api-key",
        base_url="http://localhost:8000",
        session_id="openai-demo"
    )

    # Create agent with memory
    agent = Agent(
        name="Assistant",
        instructions="You have persistent memory.",
        tools=[memory]
    )

    # First conversation: store info
    result = await Runner.run(agent, input="Save that my favorite color is teal and I work on AI.")
    print("Session 1:", result.final_output)

    # Second conversation: recall info (same session_id)
    result2 = await Runner.run(agent, input="What do you know about me?")
    print("Session 2:", result2.final_output)

    # You can also manually store/recall
    await memory.store(key="project", value={"name": "Memory Bridge"})
    data = await memory.recall(key="project")
    print("Manual recall:", data)

if __name__ == "__main__":
    asyncio.run(main())

How It Works

The MemoryBridgeTool exposes two function calls to the agent:

  • store_memory(key, value) — Saves a fact or piece of information. The agent calls this when it learns something worth remembering.
  • recall_memory(key, query=None) — Retrieves stored memories. Supports both exact key lookup and semantic search via the query parameter.

The agent framework handles the tool call loop automatically — you just register the tool and define your agent's instructions.

Multi-Session Persistence

Key design features of the OpenAI Agents SDK integration:

  • Session-scoped memory — Use session_id to isolate memory per user or conversation
  • Cross-session recall — Use the same session_id across agent restarts for continuity
  • Global memory pool — Omit session_id to share memory across all sessions (use with project for scoping)
  • Automatic tool invocation — The agent decides when to store or recall based on your instructions

⚠️ Production Tip: Store user-specific information (preferences, context) under distinct session_id values. Use a shared project for global knowledge that all agents should access.
