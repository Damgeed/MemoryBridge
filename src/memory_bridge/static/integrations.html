🚀 Quick Start

Get Memory Bridge Running in Seconds

Use our hosted instance — zero setup, just add your API key.

# Add to your Authorization header
$ curl -X POST https://memory-bridge-app-production.up.railway.app/memories \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"session_id":"my-session","agent_id":"henry","key":"hello","value":"Hello Memory Bridge!"}'

Run your own instance locally or on any server with Docker.

# Pull and run (SQLite by default)
$ docker run -p 8000:8000 memorybridge/memory-bridge
# Or with docker-compose (PostgreSQL + Redis)
$ docker compose up

Install as a Python package — use it as a library or CLI.

# Install the SDK
$ pip install memory-bridge
# Or use it as a Python library
>>> from memory_bridge_client import Client
>>> async with Client(url="https://your-instance.com", api_key="...") as mb:
ctx = await mb.query_memories(agent_id="henry", tags=["profile"])

Use the MCP server — works with Claude Desktop, Cursor, Windsurf, and any MCP-compatible client.

# Add to claude_desktop_config.json
{
"mcpServers": {
"memory-bridge": {
"command": "npx",
"args": ["-y", "@memory-bridge/mcp"],
"env": { "MEMORY_BRIDGE_API_KEY": "YOUR_API_KEY" }
}
}
}
