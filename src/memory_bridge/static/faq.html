Frequently asked questions

594|

Everything you need to know about Memory Bridge.

595|
596| 597|
598| 📋 How do I sign up? Do you store my password?▸ 599|
600| No passwords. Type your email → get a 6-digit code → enter it. That's it.

601| Or sign in with Google, Apple, or Phone (SMS) — we never see or store your password because there isn't one.

602| Forgot your password? Doesn't exist. Every login sends a fresh code. 603|
604|
605| 606|
607| 🔐 Is my instance secure?▸ 608|
609| Yes. API calls are authenticated via JWT or API keys with role-based access. Rate limiting per tier protects against abuse.

610| Your database is yours alone — multi-tenancy via schema-per-tenant with Row-Level Security. No cross-tenant data leaks. 611|
612|
613| 614|
615| 🧩 What LLMs and frameworks do you support?▸ 616|
617| Any LLM with an API — OpenAI, Claude, Gemini, Ollama, llama.cpp, OpenRouter.

618| Any framework — LangGraph, AutoGen, CrewAI. Memory Bridge sits between your code and the LLM, so the integration pattern is the same regardless of what you're using. 619|
620|
621| 622|
623| 🔄 Can I self-host?▸ 624|
625| Yes! Memory Bridge is MIT licensed. Self-host with Docker Compose on any machine — no limits, no restrictions.

626| Or use the hosted version and let us handle the infrastructure. Either way, your data stays yours. 627|
628|
629| 630|
631| 💾 How is data stored and retained?▸ 632|
633| Every memory is stored as a row in your own database. Export, query, delete, or audit at any time — it's your data.

634| Retention depends on your plan: Free (7 days), Starter (90 days), Pro (365 days). Memories older than your retention window are automatically cleaned up. 635|
636|
637| 638|
639| 🏢 Is there an Enterprise plan?▸ 640|
641| Yes — dedicated infrastructure, SSO/SAML, custom retention policies, SLA guarantees, and a dedicated support engineer.

642| Contact us for custom pricing. 643|
644|
645| 646|
647| 🛠️ Do I need to register to try it?▸ 648|
649| Nope! Open the Playground with the demo key and start storing memories immediately.

650| When you're ready to go to production, sign up for a free account to get your own dedicated API keys and higher rate limits. 651|
652|
653| 654|
655| 📦 What's included in the pricing plans?▸ 656|
657| All plans include the full Memory Bridge API — there are no feature restrictions. Plans differ by capacity: memories stored, sessions, API keys, retention period, and rate limits.

658| See the Pricing page for full details. Upgrade or downgrade anytime — changes apply immediately. 659|
