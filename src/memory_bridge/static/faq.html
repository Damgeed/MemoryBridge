Frequently asked questions

Everything you need to know about Memory Bridge.

📋 How do I sign up? Do you store my password?▸
No passwords. Type your email → get a 6-digit code → enter it. That's it.

Or sign in with Google, Apple, or Phone (SMS) — we never see or store your password because there isn't one.

Forgot your password? Doesn't exist. Every login sends a fresh code.
🔐 Is my instance secure?▸
Yes. API calls are authenticated via JWT or API keys with role-based access. Rate limiting per tier protects against abuse.

Your database is yours alone — multi-tenancy via schema-per-tenant with Row-Level Security. No cross-tenant data leaks.
🧩 What LLMs and frameworks do you support?▸
Any LLM with an API — OpenAI, Claude, Gemini, Ollama, llama.cpp, OpenRouter.

Any framework — LangGraph, AutoGen, CrewAI. Memory Bridge sits between your code and the LLM, so the integration pattern is the same regardless of what you're using.
🔄 Can I self-host?▸
Yes! Memory Bridge is MIT licensed. Self-host with Docker Compose on any machine — no limits, no restrictions.

Or use the hosted version and let us handle the infrastructure. Either way, your data stays yours.
💾 How is data stored and retained?▸
Every memory is stored as a row in your own database. Export, query, delete, or audit at any time — it's your data.

Retention depends on your plan: Free (7 days), Starter (90 days), Pro (365 days). Memories older than your retention window are automatically cleaned up.
🏢 Is there an Enterprise plan?▸
Yes — dedicated infrastructure, SSO/SAML, custom retention policies, SLA guarantees, and a dedicated support engineer.

Contact us for custom pricing.
🛠️ Do I need to register to try it?▸
Nope! Open the Playground with the demo key and start storing memories immediately.

When you're ready to go to production, sign up for a free account to get your own dedicated API keys and higher rate limits.
📦 What's included in the pricing plans?▸
All plans include the full Memory Bridge API — there are no feature restrictions. Plans differ by capacity: memories stored, sessions, API keys, retention period, and rate limits.

See the Pricing page for full details. Upgrade or downgrade anytime — changes apply immediately.
